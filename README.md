# lobo-web-site
lobo商品评价系统web端，底层的评价服务调用的项目https://github.com/beanmoon/lobo实现的http api。



项目是JDS五期一班四组在编程马拉松比赛中实现的一小部分，感谢组内所有的小伙伴。



### 核心依赖库

- java（1.7+）
- jersey（2.6）
- velocity（1.7）
- commons
- tomcat（7.0+）



### 部署

将javaweb应用打包成*.war，然后导入到tomcat下webapp的文件夹，启动tomcat，服务启动成功。



**访问链接：**
